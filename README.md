## Quick Start

### 1. Install globally (once)

```bash
cd ~/apex-framework
chmod +x install-user-level.sh
./install-user-level.sh
```

### 2. Apply to any project (one command)

```bash
cd ~/your-project
~/apex-framework/apex-init-project.sh
```

### 3. Launch

```bash
claude
```

## What You Get

### User-level (global — all projects)

- **13 universal skills** — code-standards, design-system, workflow-enforcer, debug, a11y, teach, and more
- **3 subagents** — code-reviewer (Sonnet), design-reviewer (Sonnet), researcher (Haiku)
- **2 output styles** — Educational + Mandalorian
- **10 hook scripts** — security, formatting, commit validation, context injection
- **StatusLine** — real-time tokens, cost, context bar, "by L.B. & Claude"

### Project-level (per project)

- **12 project skills** — prd, architecture, research, qa, security, performance, deploy, commit, changelog, init, e2e, cicd
- **4 path-based rules** — testing, sql, api, components
- **2 git hooks** — pre-commit (lint + secrets scan), commit-msg (conventional commits)
- **Sandbox + permissions** — defense in depth
